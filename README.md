# OMG Demons!

> Dodge arrows to escape hell

This game was created for [Mini Jam 120: HellÂ²](https://itch.io/jam/mini-jam-120-hell)

## Instructions

You are an angel trapped in hell. You don't have any means to attack the demons. To kill then you must dodge their arrows.

| key          | action |
|--------------|--------|
| wasd, arrows | walk   |
| space        | dash   |
| esc, p       | pause  |
| r            | reset  |
